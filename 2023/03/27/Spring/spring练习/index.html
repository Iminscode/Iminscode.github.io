1.1 Spring环境搭建步骤

  • 创建工程（Project&Module）
  • 导入静态页面（见资料jsp页面）
  • 导入需要坐标（见资料中的pom.xml）
  • 创建包结构（controller、service、dao、domain、utils）
  • 导入数据库脚本（见资料test.sql）右键test数据库，点SQL Scripts，点run SQL scripts，选择电脑里要执行的文件。
  • 创建POJO类（见资料User.java和Role.java）对应标准的JavaBean
  • 创建配置文件（applicationContext.xml、spring-mvc.xml、jdbc.properties、log4j.properties、web.xml)
    • 在web.xml中，需要配置全局的初始化参数，配置Spring的监听器，配置spring前端控制器
    • 在spring-mvc中，需要配置mvc注解驱动，配置视图解析器，配置静态资源访问权限
    • 在application.xml中，需要配置jdbc.properties加载数据源，配置数据源对象，配置jdbc模板JdbcTemplate

2.2 角色列表的展示步骤分析

  • 点击角色管理菜单发送请求到服务器端（修改角色管理菜单的url地址）
  • 创建RoleController和showList()方法
  • 创建RoleService和showList()方法
  • 创建RoleDao和findAll()方法
  • 使用JdbcTemplate完成查询操作
  • 将查询数据存储到Model中
  • 转发到role-list.jsp页面进行展示
