github+picgo图床设置

Author: Aiming Date: March 18, 2023  15:08:43 Category: Configuration

一 【Github+PicGo】Github图床设置并上传图片

==点击本教程参考连接==

1、创建一个github账号，登录该账号。新建一个存储库

2、下载PicGo，下载地址

  • 打开PicGo设置，把时间戳重命名打开（同一张图片重复上传时，不会失败）；开启上传提示（可打开也可不打开）

  • 打开github图床设置。

  • 中间设定token中的token的获取。打开github，点击个人头像，打开settings设置，找到developer settings设置打开，点击personal access tokens。在note填写你的用途（比如save image)，在expiration选择token的期限，可以选择永久。必须勾选repo。最后生成你的token，生成后复制你的token，一定要复制保存，他只会显示这一次，后续查找不到。

  • 设定自定义域名。

    • 不填写，加载速度会有点慢，可能会导致你typora的显示很慢甚至显示不出来，但是在博客转存的时候好像没什么问题
    • CDN加速：https://cdn.jsdelivr.net/gh/GitHub账户名/仓库名，这个加速国内似乎已经访问不了了，大家可以试试
    • fastly加速：https://fastly.jsdelivr.net/gh/GitHub账户名/仓库名，我使用这个在Typora显示图片没问题，但是在转存的时候大部分图片都是转存失败
    • gcore加速：https://gcore.jsdelivr.net/gh/GitHub账户名/仓库名 ，这个使用过程中还是有少量的图片转存失败

    如果这三个加速不行，可以尝试在链接末尾添加@main，举例CDN加速：https://cdn.jsdelivr.net/gh/GitHub账户名/仓库名@main，这样还是不行的话就只有另求他法了。

3、typora设置

  • 打开软件，点击文件，打开typora偏好设置，选择图像

  • 等待通知是否上传成功。
